#### 题目描述

给定一个排序链表，删除所有重复的元素，使得每个元素只出现一次。

**示例 1:**

```
输入: 1->1->2
输出: 1->2
```

**示例 2:**

```
输入: 1->1->2->3->3
输出: 1->2->3
```

#### 题目解析

很简单的链表基础题。

最常见的解法莫过于取得当前节点指针，与之后的节点的值比较确定是否为重复节点。是则更改当前节点的`next`指针，以便跳过下一个节点，并指向下一个节点之后的节点。

**复杂度分析：**

- 时间复杂度：*O*(n)，因为列表中的每个结点都检查一次以确定它是否重复，所以总运行时间为 *O*(n)，其中 `n` 是列表中的结点数。
- 空间复杂度：*O*(1)，没有使用额外的空间。
